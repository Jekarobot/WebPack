# WebPack 🚀

Проект демонстрирует базовую настройку и использование Webpack для сборки фронтенд-приложения. Включает обработку JavaScript, CSS и HTML.

## Технологический стек 🛠️

- **Webpack** — сборщик модулей
- **JavaScript (ES6+)** — основной язык
- **CSS** — стилизация
- **HTML** — разметка

## Установка и запуск ⚙️

1. Клонируйте репозиторий:
   ```bash
   git clone <repository-url>
   cd webpack
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите сборку:
   ```bash
   npm run build
   ```

4. Откройте `dist/index.html` в браузере.

## Примеры использования 📖

### Разработка

Для запуска в режиме разработки с автоматической пересборкой:
```bash
npm run dev
```

### Сборка для продакшена

```bash
npm run build
```

Результат будет в папке `dist/`.

## Структура проекта 📂

```
📁 webpack/
  📁 dist/               # Собранные файлы
    📄 bundle.js         # Скомпилированный JS
    📄 index.html        # Скомпилированный HTML
    📄 main.css          # Скомпилированный CSS
  📁 src/                # Исходный код
    📁 css/
      📄 style.css       # Стили
    📁 js/
      📄 app.js          # Модуль приложения
    📄 index.html        # Шаблон HTML
    📄 index.js          # Точка входа JS
  📄 package.json        # Зависимости и скрипты
  📄 package-lock.json   # Фиксация версий зависимостей
  📄 webpack.config.js   # Конфигурация Webpack
```
